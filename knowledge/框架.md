# A股短线分析框架

> 本文档是短线分析的核心知识库，包含用户的交易认知体系、量化选股规则、分析方法论。
> 复盘/开盘分析时主动读取此文件。

## 1. 交易规则速查

**涨跌停幅度**

| 板块 | 普通股 | ST 股 | 新股前5日 |
|------|--------|-------|----------|
| 主板 | ±10% | ±5%（拟调至±10%） | 不设限（30%/60%临停） |
| 创业板 | ±20% | ±20% | 不设限 |
| 科创板 | ±20% | ±20% | 不设限 |

**交易制度**
- A 股全板块 **T+1**
- 主板价格笼子 ±2%（挂单价不能偏离最优价太远）

**异动监管（股票交易异常波动）**
- 10个交易日内涨幅达100% → 进入监管，持续10个交易日，单账号每分钟限买1000万
- 30个交易日内涨幅达200% → 进入监管，持续10个交易日，单账号每分钟限买500万

**集合竞价**
- 9:15-9:20 可撤单（价格不可信，主力常造假象）
- 9:20-9:25 不可撤单（真实意愿）
- 14:57-15:00 尾盘集合竞价不可撤单

**交易时间**
- 集合竞价 9:15-9:25 → 连续竞价 9:30-11:30 / 13:00-14:57 → 尾盘集合竞价 14:57-15:00

## 2. 用户短线认知体系

**龙头的定义：第一 + 唯一**
- **第一**：涨幅第一（周期内涨幅最高）
- **唯一**：在某个关键维度上是唯一的，比如唯一破除监管的、唯一站上某个高度的、唯一穿越周期的
- 例：华电辽能是唯一破除监管并站上8板的股票，且涨幅最高，所以是龙头
- 光有"第一"不够（涨幅第一但没有独特性），光有"唯一"也不够（有独特性但涨幅不够），必须两者兼备

**主线判断标准**
1. **连续性**：主线题材一定会经历"主升→分歧→再主升"，经历过分歧仍能回来的才是真主线。只涨一天的不算主线。
2. **板块内部结构完整**：
   - **龙头**：第一+唯一，市场公认最强标的，周期内涨幅最高且具备独特辨识度（可以是连板也可以是趋势股）
   - **中军**：成交量大，板块内有地位（地位 = 涨跌能影响板块其他股，龙头也必须有地位才是真龙头）
   - **弹性**：20cm 标的（创业板/科创板），当龙头高位进不去时资金去 20cm 套利
   - 只有龙头没有中军和弹性 = 板块没有深度，不算真主线

**破局龙**
- 大多数股票面临监管都不敢过。能突破监管且不大跌的股票就是**破局龙**
- 破局龙出现后一般会有新周期，同时产生补涨（畏高资金做低位套利）
- 新的主升周期可能是破局龙自身板块，也可能是其他板块

**节点股**
- 当前龙头**跌停**或**爆量**时，产生一个"节点"——标志着当前周期结束
- 此时会对应一个新的低位股票崛起，这个股票就是**节点股**，即下一个周期的起点
- 例：3/16 豫能控股跌停 → 华电辽能一字板 → 华电辽能成为下个周期的节点股
- 节点股的识别是龙头切换/周期切换的关键信号

**三板组**
- 低位连板中有一类股票是"三板组"做的，手法固定，属于特定游资操盘
- 博主复盘中通常会点明哪些股票是三板组手法
- **这类股票不关注、不参与**，规避即可
- 已知案例：王力安防

**超预期股识别**
- **断板反包**：连板股昨日断板，今日低开理应补跌，但截止早盘拉升并保持红盘
- **跌后反转**：昨日大跌（>7%），今日低开理应持续补跌，但能翻红并冲击涨停（>7%）
- **逆势抗跌**：板块整体大跌，但该股逆势翻红或明显强于板块均值（偏离 >5%）
- **缩量强势**：竞价缩量但开盘后快速拉升放量，说明卖盘枯竭、主力控盘
- 超预期股往往是下一阶段的辨识度标的，值得重点关注

**套利规则（正股无法买入时的替代策略）**
1. **转债套利**：正股带可转债时，转债可转股，存在关联逻辑可套利
2. **20cm 弹性套利**：同板块关联性强的创业板/科创板股票，20% 涨停空间更大，踏空资金首选
3. **同板块主板套利**：同板块关联性强的主板股票（例：常山铝业涨停 → 云铝股份，同为金属铝资源股）

**板块内个股走势主动判断**
- 涨得多、涨得快，对比板块内其他个股明显有主动拉升
- 走势主动的个股往往是板块内的潜在龙头或中军

## 3. 量化选股规则（开盘分析用）

以下是用户定义的量化选股筛选条件，用于每日开盘分析 Agent（09:26）自动筛选：

**高开过顶**（全市场筛选）
1. 过去 7 个交易日内有涨停
2. 昨天不是涨停
3. 今日开盘价 > 过去 7 个交易日的最高价（创7日新高）

**断板反包**（全市场筛选）
1. 前天涨停
2. 昨天没有涨停（断板）
3. 今天高开

**板块总结**（基于股票池）
1. 统计各板块加权均涨幅，⭐辨识度股权重 ×2
2. 识别批量高开/低开的板块
3. 结合当日及近两天新闻，分析高开板块的驱动原因

## 4. 每日分析维度

### 4.1 市场情绪
- **涨停数量**：涨停板数量反映市场做多意愿（>80 强势，40-80 正常，<40 弱势）
- **跌停数量**：跌停板数量反映恐慌程度
- **涨跌比**：上涨家数 vs 下跌家数
- **炸板率**：涨停后打开的比例，反映资金接力意愿
- **连板高度**：最高连板数，反映市场投机空间

### 4.2 板块轮动
- 当日涨停最多的板块 → 主线方向
- 连续 2-3 天领涨的板块 → 确认主线
- 板块内个股梯队（龙头 / 补涨 / 跟风）
- 板块间联动关系（如"锂电 → 储能 → 电力"传导链）

### 4.3 资金流向
- 主力资金净流入板块
- 龙虎榜数据（机构 vs 游资席位）
- 北向资金动向

### 4.4 龙头股跟踪
- 辨识标准：板块首板 / 最高板 / 成交额最大 / 封单最强
- 生命周期：启动 → 加速 → 分歧 → 一致 → 衰退
- 龙头倒下后的补涨机会

## 5. 情绪周期模型

```
冰点 → 修复 → 升温 → 高潮 → 分歧 → 退潮 → 冰点
```

| 阶段 | 涨停数 | 连板高度 | 特征 | 策略 |
|------|--------|---------|------|------|
| 冰点 | <30 | 无连板 | 大面频出，恐慌 | 观望或试探低吸 |
| 修复 | 30-50 | 2-3板 | 超跌反弹，试探 | 关注率先反弹板块 |
| 升温 | 50-70 | 3-5板 | 主线浮现，赚钱效应回升 | 跟随主线龙头 |
| 高潮 | >80 | 5板+ | 板块普涨，一致看多 | 注意风险，不追高 |
| 分歧 | 50-70 | 高位股分歧 | 龙头炸板，补涨活跃 | 做补涨或等确认 |
| 退潮 | 下降 | 连板断裂 | 亏钱效应扩大 | 减仓，等待冰点 |

## 6. 复盘模板

每日复盘应回答：
1. 今天市场处于情绪周期的哪个阶段？
2. 主线是什么？有没有新的方向？
3. 龙头股表现如何？（封板强度、成交额、炸板情况）
4. 明天可能怎么走？（持续 / 分歧 / 转向）
