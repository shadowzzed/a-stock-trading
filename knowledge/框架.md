# A股短线分析框架

> 核心知识库。五博主体系精华 + 回测验证。复盘/分析时读取。

## 1. 情绪周期（底层操作系统）

所有操作围绕「当前处于情绪周期哪个阶段」展开，不做基本面分析。

```
冰点 → 修复 → 升温 → 高潮 → 分歧 → 退潮 → 冰点
```

| 阶段 | 涨停数 | 连板 | 操作 | 仓位 |
|------|--------|------|------|------|
| 冰点 | <30 | 无 | 只做辨识度超预期破局，或空仓 | 0-1成 |
| 修复 | 30-50 | 2-3板 | 关注率先反弹板块，试探 | 1-3成 |
| 升温 | 50-70 | 3-5板 | 跟随主线龙头 | 3-5成 |
| 高潮 | >80 | 5板+ | 只出不进，兑现利润 | 减仓 |
| 分歧 | 骤降 | 高位分歧 | 做最强低吸，或等确认 | 1-3成 |
| 退潮 | 持续降 | 断裂 | 空仓保命 | 0 |

**三层嵌套周期**（穿越复盘）：
- 大周期：主线行情完整周期（破局→主升→补涨→轮动）
- 中周期：一只连板龙的全部高度
- 小周期：日级别情绪波动
- 小周期在大周期拐点时可能失效

**周期三锚定点**：
1. 破局龙出现 → 新周期开端
2. 总龙涨速减缓 → 主升结束，补涨开始
3. **总龙跌停 → 周期彻底结束，必须空仓**

## 2. 竞价决定 80% 的操作

竞价是市场情绪的 X 光片，10 分钟决定全天操作基调。

**竞价三信号法**（作手阿意）：
- 价格信号：龙头高开 3%+ 且量能不缩 = 强势延续
- 量能信号：竞价缩量/放量反映主力意愿
- 联动信号：板块多只同步高开 = 方向确认

**竞价规则**：
- 9:15-9:20 可撤单（价格不可信，主力常造假象）
- **9:20-9:25 不可撤单（真实意愿）**
- 竞价超预期 → 积极参与
- 竞价不及预期 → 持仓立即卖出，新仓放弃

## 3. 选股：辨识度第一

> 回测验证：辨识度高的标的（广生堂 +4.39%、力诺药包 +8.01%）vs 低辨识度（0%）差异显著

**辨识度 = 身位 + 逻辑正宗 + 资金认可**
- 身位：板块涨幅最高、连板最高、或唯一破局
- 逻辑正宗：题材最正宗的标的（电力就做电力龙头，不做擦边）
- 资金认可：早盘封板、量能充足、席位干净

**龙头定义：第一 + 唯一**
- 第一：周期内涨幅最高
- 唯一：某关键维度唯一（唯一破监管、唯一穿越周期等）
- 两者必须兼备

**主线判断**（穿越复盘）：
- "无中军不主线，无主线不龙头"
- 板块画像：连板龙 + 中军龙 + 助攻龙 + 跟风，四角色齐全才是真主线
- 连续 2-3 天领涨 + 经历过分歧仍能回来 = 确认主线

**选股五要素排序**（空空龙）：辨识度 > 图形审美 > 板块强度 > 分时超预期 > 业绩

## 4. 买卖核心节奏

**买在分歧拐点，卖在一致高潮**：
- 冰点日 → 做辨识度超预期（仅限破局龙级别）
- 分歧日 → 做最强品种低吸
- 修复/升温日 → 跟随主线龙头
- 高潮日/一致加速 → 兑现卖出

**预期管理三原则**（作手阿意）：
- 超预期 → 买入/加仓（竞价/开盘强于预期）
- 符合预期 → 看反馈（走势符合预案）
- 低于预期 → 卖出（竞价/开盘弱于预期）

**T+1 铁律**：
- 每笔买入都是隔夜赌注，买入前必须想好明天怎么卖
- 竞价是持仓者第一个卖出窗口，竞价不及预期 = 立即卖出
- 满仓 = 丧失明天机动权，**至少留 30% 现金**
- 退潮期买入 = T+1 下无法当日止损，只能被动挨打

## 5. 退潮预警与空仓

**一叶知秋三重验证**（雪球博主）：
1. 5 板+ 高标出现首个跌停/大阴 = 信号一
2. 中位股负反馈（3-4 板集体炸板）= 信号二
3. 竞价集体不及预期（龙头+中军+弹性全部低开）= 信号三
- 三重同时触发 → 强制空仓

**空仓硬触发条件**（满足任一即空仓）：
- 周期龙/总龙跌停
- 变盘日确认（绿盘 >4000 家 + 缩量）
- 量能 <2.0 万亿
- 跌停 >=15 家 + 炸板率 >50%
- 高标集体跌停（5 板以上多只同时跌停）

> 回测验证：冰点期推荐标的命中率 0%，空仓才是正确操作

## 6. 禁令清单（硬性约束，违反必亏）

| 禁令 | 原因 |
|------|------|
| 一字板不追 | 买不进是常态，强行追入大亏根源 |
| 竞价不买入 | 竞价信息不充分，等开盘确认（竞价卖出除外） |
| 补涨股不碰 | 接别人先手筹码，高度有限 |
| 偷鸡板不做 | 下午封板是套利行为，真主动早上就拉 |
| 退潮期不做弱转强 | 退潮期弱转强多为骗炮 |
| 高潮期（>7板）只出不进 | 极端高潮次日大概率分歧退潮 |
| 追高连板股谨慎 | 连续涨停 2 板以上追高风险极大，推荐必须降仓位至 2 成以下 |
| 三板组不参与 | 特定游资手法，散户必亏 |

## 7. 败笔检查清单（延边刺客）

每个交易标的过败笔检查，发现任一即放弃：

| 类型 | 含义 |
|------|------|
| 竞价败笔 | 该一字的没一字，该高开的低开 |
| 发酵败笔 | 板块一字板方向偏离预期 |
| 身位败笔 | 同板块有更高身位、更早封板的竞争对手 |
| 个股败笔 | 历史封板常炸板、基本面硬伤 |
| 主动性败笔 | 被带动才涨，缺乏自主拉升 |

## 8. 量化参考

**情绪指标**：
- 涨停数：>80 强势，40-80 正常，<40 弱势
- 炸板率：<60% 坚决，70-80% 分歧，>80% 风险大
- 连板高度：市场投机空间的温度计

**量能门槛**：
- 全市场：>=2.5 万亿容错率高，<2.2 万亿持续性差，<2.0 万亿极度缩量
- 个股最低成交额：小盘(<50亿) >=3亿，中盘(50-200亿) >=5亿，大盘(>200亿) >=10亿

**发酵等级**（延边刺客）：
- 仅龙头涨停 = 无发酵（一日游）
- 1-3 只跟随 = 轻度（试探）
- 5 只以上 = 充分（板块效应确立）
- 10 只以上 = 过度（可能见顶，"毕业照"）

## 9. 超预期股识别

- **断板反包**：连板股昨日断板，今日低开后拉升翻红
- **跌后反转**：昨日大跌 >7%，今日翻红冲击涨停
- **逆势抗跌**：板块大跌但该股翻红，偏离 >5%
- **缩量强势**：竞价缩量但开盘后快速放量拉升

## 10. 交易规则速查

- A 股 T+1，可转债 T+0
- 主板 +-10%，创业板/科创板 +-20%，北交所 +-30%
- 新股前 5 日不设涨跌幅（盘中临停 30%/60%）
- 主板价格笼子 +-2%（挂单价不能偏离最优价太远）
- 10 日涨幅 100% 或 30 日涨幅 200% → 异动监管
- 集合竞价：9:15-9:20 可撤单，9:20-9:25 不可撤单，14:57-15:00 尾盘竞价不可撤单
